# Security policy

Vettore è una demo pensata per girare **solo in locale** (la porta è pubblicata su `127.0.0.1` e non c'è autenticazione reale): non esporla su reti non fidate.

Se trovi una vulnerabilità, segnalala in modo riservato tramite
[**Security → Report a vulnerability**](https://github.com/It-Energy-Ai/Logistica_Gas_Natural_Up_Stream/security/advisories/new)
invece di aprire una issue pubblica. Grazie!
